## AI智能命名
- 默认值： 无

- 作用：开启后F2C转码后将使用GPT进行图层翻译，将图层的中文名称转化为英文变量带入到代码中
> 注意该设置仅内部使用，外网用户请选择`无`

- 作用：根据需要选择生成代码的开发语言
