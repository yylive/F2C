连接上 **f2c-server** 后，将会看到如下页面

![代码面板](./image/code.png)

- 标签按钮可以为该图层添加标签信息，详细内容见进阶标签用法文档。
- 参数设置为你选择的代码类型和相关配置，具体见[配置](./setting.md)。
- 输出目录和图层名字均用于**f2c-server**在本地文件生成的目录和文件夹名，可以根据自己的意愿修改。另外，图层命名可以使用AI生成（供内部使用），AI将识别改图层的具体信息来给图层附上语义化的名称。
