## 什么是DSL

DSL(Domain-specific language)的全程是领域特定语言。在F2C转码过程中，我们把`设计语言` 通过`转换器实现` 转译为 `特定代码语言`这个过程中的`转换器实现`，简称为DSL。

## 我们支持哪些 DSL

目前我们支持了`React`, `React Native(RN)`, `鸿蒙`，`Taro`, `Vue`这些DSL，通过选项面板可以选择我们想要生成的端代码。
